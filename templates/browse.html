🗂 Help & Guide

What is Filebrowser?

Filebrowser lets you browse, upload, download, and manage files on the server from any web browser. Think of it as a remote file manager.

Navigation

Use the breadcrumb bar below the header to jump to any parent folder. Click the ~ (home) segment to return to the root.

Click any folder name in the file list to open it. Use the .. entry at the top of the list to go up one level.

Uploading Files

Pick files with the upload area at the bottom, then press Upload. Large files are sent in chunks — you can pause, resume, or cancel transfers in the Transfers panel.

If an upload is interrupted (browser closed, network drops), come back to the same folder and click ▶ Resume on the partial entry to continue where you left off.

Downloading Files

Click the ⬇ button next to any file. The download streams in the background — progress appears in the Transfers panel. Once complete the file saves automatically.

Deleting Files

Click the ✕ button. You will be asked to confirm before the file or folder is permanently removed. System files (the app binary and .env) are protected.

Tips

  • Bookmark your most-used folders — the URL updates with ?path=… so you can copy it.
  • File size and modification-date columns appear on wider screens.
  • File-type icons help you spot what you're looking for at a glance.
